Pluto

A CSS framework with built-in animations.

Features

Simple

Pluto uses very intuitive class names to allow for quick and easy setup.

Fast

Pluto's small size (only 8KB) means that pages load as soon as possible.

Lively

Pluto's built-in animations means spending more time on building.

Tiles

Tiles are similar to columns. They are created using the tiles class and children should have the tile class. Each tile can have an h4 with a tile-title class and a p with a tile-text class.

Code

Small pieces of code can be written inside of text with the <code> tag, but larger sections of code work better when inside of <pre> and <code> tags.



    <!-- Structured Code -->
    <pre><code>
      .col-1 {
        flex-grow: 1;
      }
    </pre><code>

Buttons

Buttons and inputs come with two style classes: button-empty and button-filled.

Anchor
Anchor 


    <!-- Empty Buttons and Inputs>
    <a class="button button-empty" href="">Anchor</a>
    <button class="button-empty">Button</button>
    <input clas="button-empty" type="submit" value="Submit Input">
    <input clas="button-empty" type="button" value="Button Input">

    <!-- Filled Buttons and Inputs>
    <a class="button button-filled" href="">Anchor</a>
    <button class="button-filled">Button</button>
    <input clas="button-filled" type="submit" value="Submit Input">
    <input clas="button-filled" type="button" value="Button Input">

Other Classes

There are a few classes that can be applied to any element to save time writting CSS.

  • has-border - Add a border

  • has-shadow - Add a box-shadow

Animations

Hover over the buttons to see some of Pluto's built-in animations.



    <!-- Animations -->
    <button class="button-filled glow">Button</button>
    <button class="button-filled bounce">Button</button>
    <button class="button-filled pulse">Button</button>

Loaders

Loaders are created within their own divs. They are created by using a combination of the loader class and a style class (loader-1 through loader-3).




    <!-- Loaders -->
    <div class="loader loader-1></div>
    <div class="loader loader-2></div>
    <div class="loader loader-3></div>
